input태그

  • type을 꼭 명시해줘야 한다 ex) input type="text"
  • placeholder는 빈칸일때 나타낼수있는 옵션이다
  • maxlength는 최대길이를 제한한다
  • minlength는 최소길이를 정해준다
  • min은 type이 number일때 최소값을 정해준다 ( max도 마찬가지 )
  • type이 file일때 accept옵션으로 파일 형식을 정해줄수있다
  • required 무조건 입력해야하는 태그임을 명시해준다
  • disabled 입력을 제한할떄 사용한다
  • value 초기값으로 설정한다
  • label을 사용할땐 for와함께 쓸것 input id를 주면된다
